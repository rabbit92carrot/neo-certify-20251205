# Changelog

네오인증서 (Neo-Certify) 프로젝트의 모든 주요 변경사항을 기록합니다.
형식은 [Keep a Changelog](https://keepachangelog.com/ko/1.0.0/)를 따릅니다.

## [1.5.0] - 2026-01-29

### Added
- 비밀번호 찾기 기능 — 사업자등록번호 + 이메일 확인 후 재설정 이메일 발송 (#31)
- 계정 찾기 기능 — 사업자등록번호 + 대표연락처로 마스킹된 이메일 확인 (#31)
- 회원가입 후 이메일 인증 안내 페이지 (`/register/verify-email`) (#32)
- 인증 메일 재발송 기능 (rate limiting 적용) (#32)

### Improved
- 로그인 시 이메일 미인증 상태 명확한 안내 메시지 + 재발송 버튼 (#32)
- 비밀번호 재설정 redirectTo URL 수정 (#31)
